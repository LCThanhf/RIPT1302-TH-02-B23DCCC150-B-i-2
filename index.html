Giới thiệu các dạng biểu đồ

1. Scatter Plot

Description: A scatter plot is a type of plot that displays values for two variables as points on a two-dimensional Cartesian plane.

2. Line Chart

Description: A line chart or line plot or line graph or curve chart is a type of chart which displays information as a series of data points called 'markers' connected by straight line segments.

3. Bar Chart

Description: A bar chart or bar graph is a chart or graph that presents categorical data with rectangular bars with heights or lengths proportional to the values that they represent.

4. Pie Chart

Description: A pie chart is a circular statistical graphic, which is divided into slices to illustrate numerical proportion.

5. Donut Chart

Description: A donut chart is a variant of the pie chart, with a hole in the center.

6. Bubble Chart

Description: A bubble chart is a type of chart that displays three dimensions of data.
